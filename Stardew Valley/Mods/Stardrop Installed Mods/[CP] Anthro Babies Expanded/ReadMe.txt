Extract this folder into your Stardew Valley/Mods directory, then run your game and load a save file with the mod installed. This will create a config.json file.
To edit this file, close your game. The config.json can be edited with notepad.
Please check the "Hairstyle Guide" folder for a detailed hairstyle guide.

1) Female Toddler Hairstyle :
"FemaleHairOne" will set the female toddler's hair as one of the two available hairstyles.
"FemaleHairTwo" will set the female toddler's hair as one of the two available hairstyles.

2) Male Toddler Hairstyles
"MaleHairOne" will set the male toddler's hair as one of the two available hairstyles.
"MaleHairTwo" will set the male toddler's hair as one of the two available hairstyles.

4) Sebastian Ginger Babies
"No" will use the default black wolf skin for Sebastian's children.
"Yes" will use ginger fox skin for Sebastian's children.
