DIY Pets for Content Patcher 2.0+
This pack for content patcher allows you to easily replace the vanilla cats & dogs available in Stardew Valley 1.6+

:: REQUIREMENTS::
• Stardew Valley 1.6+
• SMAPI up-to-date
• Content Patcher up-to-date

:: FEATURES::
• Changes the sprites of vanilla cats & dogs in-game
• Changes the displayed pet's faces in the pause menu

:: CREDITS::
♦ LavenderLight for inspiring this pack with their DIYCP series
♦ Pathoschild for Content Patcher

:: DIRECTIONS::
• Unzip the folder '[CP] DIY Pet Replacer' into your 'Stardew Valley/Mods/' folder
• In the 'assets' folder, copy an image of the pet you wish to change with into the folder, and rename that file to the pet you are replacing (included are the default animals as placeholders).
　♦ EX: you downloaded a file called 'cutecat.png'; copy that file into the 'assets' folder and rename it to 'cat1.png' to replace the second cat (cat for first cat, cat2 for third cat; this is because ConcernedApe, the developer, has named them this way in the game's 'Content/Animals/' folder, so I followed his naming convention).
• Do the same thing for the 'catcursor.png' file to get your changed cat to show in the pause menu.
• Startup the game with SMAPI and a config file will be automatically generated. You can edit this in case you don't want to change the horse, or you are using a different mod to change the dogs, for example, by disabling each animal you do not wish to alter.
• Enjoy! Please feel free to message me on Nexus if you have any questions!!
