Hey!

1.) If you've already installed this game from 2.0.0 and on, and have played it, you can delete "config.json" if you want, so you can keep your current configuration, avoiding defaults.

2.) Likewise, if you've ever editted custom parts, don't let it replace your hardwork! Always make backups of your custom body, feet, shoes, etc . . . so you can add them back if you so choose.

3.) Install this mod into Stardew Valley/Mods, don't touch anything else. Use Custom Furry Farmer (downloaded seperately, read it's readme as well) for your edits, and use "custom" in-game to try it out.


Please post bugs on the nexusmod page, and feel free to submit ideas or DM submissions directly to Krystedez on nexusmods or Discord (same name in both areas)
